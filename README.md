# CareerMaze
PROJECT FLOW:
Level 1:

START → Initialization → Login/Explore Branch


Level 2: 

- Branch A: Login → Validate → Set Flag
  
- Branch B: Explore → Skip to Main Menu

  
Level 3:

Main Menu → 5 Options (Quiz, Explore, Add Admin, Delete Admin, Exit)


Level 4:

- Quiz → Load → Display Questions → Score → Recommend Stream → Navigate from 10th
  
- Explore → Choose Start Point → Navigate Tree

- Add Admin → Input → Insert → Save
  
- Delete Admin → Input → Delete → Save
  
- Exit → End

  
Level 5 (Inside Navigate):

Show Current Node → Show Children → User Choice →

- Child Number → Go Deeper
  
- 0 → Go Back
  
- -1 → Undo Delete
  
- -2 → Search Career
  
- -3 → Delete (Session)
  
- -4 → Add (Session)
  
- -5 → Delete (Permanent) → Save
  
- -6 → Add (Permanent) → Save
  
- -9 → Add Quiz Question → Append to File
  
- -10 → Print Path → Exit
